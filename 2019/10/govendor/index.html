Govendor

十月 28, 2019 发布在 docs

安装govendor

go get -u -v github.com/kardianos/govendor
运行 govendor 检测安装结果
govendor

初始化

govendor init

收集依赖包

go add +external

安装部署govendor项目

进入git clone完成后的项目目录下
govendor sync
不过貌似也不需要，只要vendor目录在项目目录下即可
